## 📌 Problem Statement
Every day, **millions of people order food online** — completely blind to what's happening in the kitchen.

| What Users See | What They Don't See |
|---|---|
| ⭐ Star ratings (taste-based) | 🦠 Hygiene scores & inspection data |
| 📸 Food photos | 🚨 Health code violations or past incidents |
| 💰 Price & delivery time | 🧪 Nutritional transparency per dish |
| 🛵 Delivery tracking | ⚠️ AI-flagged risk indicators |

> **The food delivery industry has a transparency crisis.** Taste ratings never tell the whole story — and that gap costs people their health.

---

## 💡 Solution — SafeBite
**SafeBite** doesn't just connect you to restaurants. It gives you the **full picture** before you order.

We surface **verified safety data**, **real nutrition metrics**, and **AI-flagged risk signals** so every food decision is an informed one — without sacrificing convenience or experience.

```
Browse → Analyze → Compare → Order → Track
```

SafeBite is built on one core belief: **your health is non-negotiable, and transparency is a right — not a privilege.**

## ✨ Key Features

### 🛡️ Safety & Intelligence
- **Verified Safety Score System** — Restaurant kitchens rated on cleanliness, inspection history, and hygiene compliance
- **Health & Nutrition Rating** — Per-dish nutritional breakdown with intelligent scoring
- **AI Risk Prediction Engine** — Machine learning flags risk indicators based on historical patterns and live signals
- **Kitchen Transparency Dashboard** — Real data on certification, last inspections, and food handling standards

### 🔍 Discovery & Comparison
- **Smart Restaurant Comparison** — Side-by-side safety + health + price analysis
- **Advanced Smart Filters** — "Only Safe Mode", "Healthy Mode", dietary preferences, allergen filters
- **Curated Safety Leaderboard** — Top-rated kitchens based on verified data, not just popularity

### 🛒 Full Ordering Flow
- **Intuitive Cart & Checkout** — Clean, frictionless ordering experience
- **Secure Payment Flow** — Multiple payment methods with order confirmation
- **Real-time Order Tracking** — Live status from preparation to doorstep delivery
- **Order History & Re-order** — Smart suggestions based on past safe & healthy orders

### 👤 User Experience
- **Personalized Dashboard** — Your health stats, safety preferences, and order trends
- **Smart Alerts & Notifications** — Real-time updates when a restaurant's safety score changes
- **Profile & Health Preferences** — Dietary goals, allergen profiles, and health targets

## ⚙️ Installation & Setup

Get SafeBite running locally in under 60 seconds.

### Prerequisites
- Node.js `v18+`
- npm `v9+`

### Steps
```bash
# 1. Clone the repository
git clone https://github.com/your-username/safebite.git

# 2. Navigate into the project
cd safebite

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

> 🌐 The app will be live at **`http://localhost:5173`**

## 📊 Why This Project Matters

This isn't a side project. It's the infrastructure layer that the food delivery industry is missing.
